# ProSubscriber – AI Powered Subscription Management App

## Overview
ProSubscriber is a full stack mobile application built to help users manage subscriptions, monitor recurring spending, and receive AI powered recommendations for smarter financial decisions.

The system integrates a React Native frontend, a Kotlin based Ktor backend, a MySQL database, and a Flask based machine learning recommendation service.

## Live Demo
Website: https://prosubscriberbyteamerudite.odoo.com/

- **Backend:** Kotlin, Ktor

## How to Run
### Frontend
1. Install dependencies  
2. Start the Expo development server  
3. Run the React Native application  

### Backend
1. Open the backend project in IntelliJ IDEA  
2. Configure the database connection locally  
3. Build the Gradle project  
4. Run the Ktor server  

### Machine Learning Service
1. Install Python dependencies  
2. Configure the local database connection  
3. Run the Flask application
