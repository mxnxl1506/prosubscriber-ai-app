# ProSubscriber – AI Powered Subscription Management App

## Overview
ProSubscriber is a full stack mobile application designed to help users track subscriptions, monitor spending patterns, and receive recommendation based insights.

The project combines a React Native frontend, a Ktor backend, a MySQL database, and a Flask based machine learning recommendation service.

## Live Demo
Website: https://prosubscriberbyteamerudite.odoo.com/

## My Contribution
This was a team project. My primary responsibilities focused on backend development and system integration, including:

- Developing backend infrastructure using Ktor in Kotlin
- Designing and implementing RESTful APIs
- Managing backend request handling, middleware, and error handling
- Integrating Firebase Admin SDK for push notifications
- Working with MySQL using Exposed ORM
- Connecting backend services with the Flask based ML recommendation service
- Supporting frontend backend integration, testing, and technical documentation

## Tech Stack
- Frontend: React Native, Expo
- Backend: Ktor, Kotlin
- Database: MySQL, Exposed ORM
- Machine Learning Service: Flask, Python
- Notifications: Firebase Admin SDK

## Key Features
- User registration and authentication
- Subscription tracking and management
- Spending visibility across subscriptions
- Recommendation features powered by machine learning
- Push notifications
- Frontend backend integration through REST APIs

## How to Run

### Frontend
1. Install dependencies
2. Start the Expo development server
3. Run the React Native application

### Backend
1. Open the backend project in IntelliJ IDEA
2. Configure the database connection locally
3. Build the Gradle project
4. Run the Ktor server

### Machine Learning Service
1. Install Python dependencies
2. Configure the local database connection
3. Run the Flask application
